How to Use

A short intro to your way around

Hide Data

You can hide any file type within any other type. As of now, we allow hiding images and text inside images.

Extract Data

If you have a file along with certain keys, you can extract and reveal the hidden data.

Security

Extraction requires keys set while hiding the data, which ensure your data's safety even if it falls into the wrong hands.
