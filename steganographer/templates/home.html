Overview

A jist of what we do here

Hide Data

You can hide any file type within any other type. As of now, we allow hiding images and text inside images.

Extract Data

You can extract and reveal the hidden data inside files.

Security

Extraction requires KEYS that are set while hiding the data, which ensure your data's safety even if it falls into the wrong hands.

How to Hide Data

  • Select Cover Image

    Select image under which you want to hide data.

  • Select Type

    Select the type of data you will hide; text or Image.

  • Select LSB

    Chose from 1, 2 or 4. Each key results in a different type of steganography, making the process more secure.

  • Share Away

    You can now share the generated image without worrying about the data (hidden inside) being discovered.

How to Extract Data

  • Obtain the Image

    Acquire the image which has some hidden data.

  • Select Type

    Select the type of data which was hidden; Text or Image.

  • Select LSB

    Select LSB specified while hidding the data.

  • Scheme Away

    The hidden data is returned. You can now use your data without worrying about someone else discovering it!!

Notes

This application was made keeping in mind the privacy concerns of every digital human. In times where data is more costly than blood, it is mandatory that we take precaution every way we can. This simplistic, minimilistic framework makes information hiding and transfer secure, fun and extremely easy, so the users don't have to worry about issues like privacy, leaking of data, etc. while sharing digital data.

Secure

Hiding/Revealing both require atleast two KEYS (Type and LSB) which ensure data security even if the file is compromised.

Better than Encryption

Steganography is better as it completely hides the existence of 'hidden' data. Combined, they are even more secure and powerful.

Easy to Use

The website is built in such a way that data hiding and revealing is as easy as clicking a button, literally. The user can easily hide, share, and extract sensitive data without a worry.

Scope

The end goal is to be able to hide any file type inside any other file type. While this goal is not too optimistic, and a lot of other conversions already exist, but their efficiency and variety are still limited.
